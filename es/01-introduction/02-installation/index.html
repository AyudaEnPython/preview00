La versión 2.7 concluyo en enero del 2020 y de la versión 3.x, la 3.9 es la siguiente en concluir su suporte (puedes comprobarlo aquí).

Nota

Tener en cuenta que instalar la última versión puede generar problemas de compatibilidad con algunas bibliotecas de terceros. Es por ello que se recomienda trabajar con entornos virtuales y un administrador de versiones como uv.

A continuación se muestran las diversas formas de instalar Python (verificar con el comando anterior la correcta instalación de Python en el sistema).

Instalador Oficial

Sección titulada «Instalador Oficial»

  1. Ir al sitio oficial de Python e ir a la sección de descargas (Downloads).

  2. Seleccionar la versión de Python adecuada (más reciente o una versión específica) de acuerdo a la arquitectura del sistema operativo (32 bits o 64 bits que cumpla los requisitos mínimos del sistema) y descargar el instalador.

    Versiones de Python para Windows
    Versiones de Python para macOS

    Opcional (plataformas soportadas) Puedes obtener mas información sobre las plataformas que CPython soporta en este enlace (PEP11).

  3. Ejecutar el instalador y esperar a que termine el proceso de instalación.

    Sistema operativo Windows

    Marcar la casilla Add python.exe to PATH durante el proceso de instalación. Esta opción agrega automáticamente la ruta de Python a las variables de entorno del sistema, lo que permite ejecutar Python y sus comandos desde cualquier ubicación.

Gestores de Paquetes

Sección titulada «Gestores de Paquetes»

Un gestor de paquetes es una herramienta que automatiza la instalación, actualización, configuración y eliminación de software. Facilita la gestión al encargarse de las dependencias de los paquetes y asegurar que se instalen todos los componentes necesarios. 1

En la siguiente tabla se puede apreciar los gestores de paquetes más populares de cada sistema operativo:

LinuxDPKG, APT, DNF, Pacman, YUM

uv

Sección titulada «uv»

uv es un administrador de proyectos y paquetes de Python extremadamente rápido, escrito en Rust.4

Si Python yá se encuentra instalado en el sistema operativo, uv lo detectará y usará sin necesidad de configuración alguna. Sin embargo, uv también puede instalar y administrar versiones de Python. uv automáticamente instalará las versiones de Python que faltan según sea necesario.

Se puede instalar uv de varias formas (Windows, macOS, Linux):

A través de un script (instalador independiente)
  • Windows
  • macOS
  • Linux
Instalar uv a través de un gestor de paquetes
  • Winget
  • Scoop
  • Homebrew
A través de Cargo
A través de PyPI
  • pip
  • Entorno aislado usando pipx
